class  Any
 Any allows us to simplify our interfaces when the type a function can consume is largely variable, or more importantly, extremely late, such as with an event system. Instead of creating tens to hundreds of member functions able to consume specific type, we generalize to to take an Any (think of it as an "anything"), then let the implementation verify the actual type and act accordingly. Detailed Description

NOTICE: The contents of this header were pulled from the Boost library, modified to remove usage of Boost, and replace that usage with C++11. This was done as Any is the last remaining feature from boost, not provided via C++11, that I like to use, and I do not want to have to continue to pull in a substantially large library for one tiny piece. All repsect to those who created an maintain this originally. I thus assume no rights, responsibility, nor credit for the contents here-in.

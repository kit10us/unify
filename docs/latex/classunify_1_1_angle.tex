\doxysubsection{Detailed Description}
An \doxylink{classunify_1_1_angle}{Angle} abstract so we can move between degrees and radians easily, and simplify math by pushing the actual conversion format to the beginning and end of the lifetime of the math -\/ that is to say, we don\textquotesingle{}t care if A + B is degrees or radians, we only care that the result is in degrees or radians.
